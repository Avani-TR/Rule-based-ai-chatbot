Rule-Based AI Chatbot

Overview
This project is a simple rule-based AI chatbot built using Python.  
It responds to user inputs using predefined logic and runs in a continuous loop.



 Features
- Handles greetings (hi, hello, hey)
- Responds to basic questions
- Continuous conversation loop
- Exit using "bye"
- Simple and beginner-friendly



 Technologies Used
- Python
- VS Code



 How to Run

1. Open terminal
2. Run the file:

```bash
python chatbot.py


 Example Interaction

You: hello  
Bot: Hello! Nice to meet you  

You: your name  
Bot: I am your AI chatbot!  

You: bye  
Bot: Goodbye! Have a nice day!
